Array Iteration

Array forEach

The forEach() method calls a function (a callback function) once for each array element.This function can take 3 parameters.

const numbers = [45, 4, 9, 16, 25];
let txt = " ";
numbers.forEach(myFunction);


function myFunction(value, index, array){
txt += value + " br/>"
}

Array map()

The map() creates a new array by performing a function on each array element.
The map() does not execute the function for array elements without values.
The map() does not change the original array.

Example:
Each array value will be multiplies by 2:
const numbers1 = [45, 4, 9, 16, 25];
const numbers2 = numbers1.map(myFunction);

function myFunction(value, index, array){
return value * 2;
}

Array flatMap()

flatMap() method first maps all the elements of an array and then creates a new array by flatenning the array.

Example:
const myArr = [1, 2, 3, 4, 5, 6];
const myArray = myArr.flatMap((x) =>x * 2);
script>
const myArr = [1, 2, 3, 4, 5,6];

document.getElementById("demo").innerHTML = newArr;
/script>

Array filter()

filter() methods creates a new array with array elements that pass a test.

Example:
const numbers = [45, 4, 9, 16, 25];
const over18 = numbers.filter(myFunction);
function myFunction(value, index, array) {
return value > 18;
}

Array reduce()

reduce() runs a function on each array element to produce (reduce it to) a single value.
This method works from left-to -right in the array.

Example:
const numbers = [45, 4, 9, 16, 25];
let sum = numbers.reduce(myFunction);
function myFunction(total, value, index, array){
return total + value;
}

Note that the functions takes 4 arguments :
  • the total (initial value, previous returned value)
  • the item value
  • the item index
  • the array itself

Array reduceRight()

reduceRight() runs a functions on each array element to produce(reduce it to) to a single value. reduceRight() works from right to left in the array.

Example:

Array every()

every method checks if all array values pass a test.
This example checks if all arrays values are larger than 18.

Example:
const numberN = [40,25, 15, 5, 35, 30];
let allOver18 = numberN.every(myFunction);
function myFunction(value, index, array){
return value > 18;
}

Note that the functions takes 3 arguments :
  • the item value
  • the item index
  • the array itself

Array some()

some() checks if some array values pass a test.
This example checks if some array values are larger than 18.

Example:
const numbersN = [40,25, 15, 5, 35, 30];
let someOver18 = numbersN.every(myFunction);
function myFunction(value, index, array){
return value > 18;
}

Note that the functions takes 3 arguments :
  • the item value
  • the item index
  • the array itself

Array from()

Array.from() returns an Array object from any object with a length property or any iterable object.

Example:
Array.from("ABCDEFG");
const myArrs = Array.from("ABCDEFG123");
document.getElementById("demo20").innerHTML = myArrs;

Array keys()

Array.keys() returns an Array Iterator object with the keys of an array.

Example:
const fruits = ["Banana", "Orange", "Apple", "Mango"];
const keys = fruits.keys();
for (let x of keys){
text += x + "br>";
}

Array entries()

Array.entries() create an array Iterator and then over the key/value pairs:

Example:
const fruits = ["Banana", "Orange", "Apple", "Mango"];
const f = fruits.entries();
for (let x of f ){
document.getElementById("demo").innerHTML += x;
}

Array with()

array with() has been added as a safe method of updating elements in an array without altering the original array.

Example:
const months = ["Januar", "Februar", "Mar", "March"];
const myMonths = months.with(2, "March")

Array Spread(...)

... operator expands an iterable(like an array) into more elements:

Example:
const q1 = ["Jan", "Feb", "Mar"];
const q2 = ["Apr", "May", "Jun"];
const q3 = ["Jul", "Aug", "Sep"];
const q4 = ["Oct", "Nov", "May"];
const year = [...q1, ...q2, ...q3, ...q4];
