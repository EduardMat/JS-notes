Array length()

The length returns the length of an array.

Example:
const fruits = ["Banana", "Orange", "Apple", "Mango"];
let size = fruits.length;
4

Array toString()

Arrat to String() converts an array to a string array values, separated by commas.

Example:
const fruits = ["Banana", "Orange", "Apple", "Mango"];
let newFruits = fruits.toString();
Banana, Orange, Apple, Mango

Array at()

The at() method returns am indexed element from an array.

const fruits = ["Banana", "Orange", "Apple", "Mango"];
let fruit = fruits.at(3);
or:
let fruit = fruits[3];

Array join()

The join() methods alsp joins all array elements into a string.
It behaves just like toString(), but in addition you can specify a separator.

Example:
const newFruits = ["Banana", "Orange", "Apple", "Mango"];
document.getElementById("demo").innerHTML = newFruits.join(" * ");

Then you work with arrays, it easy to remove elements and add new elements.
The pop is getting out items from an array, and the push is adding items into an array.

Array pop()

The pop() methods removes the last element from an array.

Example:
const otherFruits = ["Banana", "Orange", "Apple", "Mango"];
otherFruits.pop();

Array push()

The push() medthods add a new element to an array - at the end.

const oldFruits = ["Banana", "Orange", "Apple", "Mango"];
oldFruits.push("Kiwi");

Shifting is equivalent to popping, but working on the first element instead of the last one.

Array shift()

The shift() method removs the first array element and "shifts" all other elements to a lower index.

Example:
const fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.shift();
const fruits = ["Orange", "Apple", "Mango"];

Array unshift()

The unshift() method adds a new element to an array (at the beggining) and unshift the older elemenets.

Example:
const fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.shift("Lemon");
const fruits = ["Lemon", "Orange", "Apple", "Mango"];

Array copyWithin()

The copyWithin() method copies array elements to another position in an array.

Example:
Copy to index 2, all elements from index 0:
const fruits = ["Banana", "Apple", "Orange", "Mango"];
fruits.copyWithin (2,0);
Banana, Apple, Banana, Mango

Array flat()

Flattening an array is the process of reducing the dimentionality of an array.
Flattening is useful when you want to convert a multi-dimensional array into a one-dimensional array.

Example:
const myArr = [[1, 2], [3,4], [5,6]];
const = myArr.flat();
1,2,3,4,5,6

Array splice()

The splice()method can be used to add new items to an array.
The first parameter (2) defines the position where elements should be added.
The second parameter (0) defines how many elements whould be removed.
The splice() returns a new array with the new elements or without the deleted ones.
Example:
const fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.splice(2, 0, "Lemon", "Kiwi");
Banana,Orange,Lemon,Kiwi,Apple,Mango

Array toSplice()

Array toSplice() has been later added as a safer method to splice an array without altering the original array.
The difference between the new and the old, is that the old alter the original while the new creates a new array.

Example:
const months = ["Jan", "Febr", "Mar", "Apr"];
const spliced = months.toSplice(0,1);
Feb,Mar,Apr;

Array slice()

The slice() method slices out a piece of an array into a new array.

Example:
Slice out a part of an array starting from array element 1 (Orange);
The (1) indicates how many elements to be removed starting from the start.
fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
const citrus = fruits.slice(1);
Orange,Lemon,Apple,Mango

The splice () can take two arguments, like (1, 3 ). The first element will be deleted and rest from the third position.

Example:
fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
const citrus = fruits.slice(1, 3);
Orange, Lemon
