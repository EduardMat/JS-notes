Array Search

Array IndexOf()

The indexOf() method searches an array for an element value and returns its position.

Example:
const fruits = ["Apple", "Orange", "Apple", "Mango"];
let position = fruits.indexOf("Apple");
Apple is found in position 0
Important!
array.indexOf(item, start)
item = this is required, is indicating the item which we search for.
start = this is optional, is idicating where to start the search.
array.indexOf() returns -1 if the items if not found.

lastIndexOf()

Array.lastIndex.Of() is the same as array.indexOf(), but returns the position of the last occurence of the specific element.

Example:
Search an array for the item "Apple":
const fruits = ["Apple", "Orange", "Apple", "Mango"];
let position = fruits.lastIndexOf("Apple");
Apple is found in position 2

Array Includes()

array.includes is allowing to check if an element is present in an array, including NaN- unlike of indexOf()

Example:
const fruits = ["Apple", "Orange", "Apple", "Mango"];
fruits.includes("Mango");
returns:true

Array find()

The find() returns the value of the first array element that passed a test function. This example finds (returns the value of ) the first element that is larger than 18.

Example:

function myFunction(value, index, array){ return value > 18;

Array findLast()

findLast()method that will start from the end of an array and return the value of the first element that satisfies a condition.

const temp = [27, 28, 30, 40, 42, 35, 30];
let high = temp.findLastIndex(x => x > 40 );
The last temperature over 40 was 42
