Array Sorting

Array Sort()

sort() method sorts an array alphabetically.

Example:
const fruits = ["Apple", "Orange", "Apple", "Mango"];
fruits.sort();
Apple,Banana,Mango,Orange

Array reverse()

reverse() list the array elements from the end to start.

Example:
const fruits = ["Apple", "Orange", "Apple", "Mango"];
fruits.reverse();
Mango,Apple,Orange,Banana

The sort() and reverse() can be used combined, as in the example bellow:
The sort() will arrange them in an alphabetical order, and the reverse will arrange them from teh z to a .

Example:
const fruits = ["Apple", "Orange", "Apple", "Mango"];
fruits.sort()
fruits.reverse();
Orange,Mango,Banana,Apple

Array toSorted()

toSorted() method as a safe way t0 sort an array without altering the original array.

toSorted() => keeps the original array unchanged and creates a new array;
sort() =>alters the existing array;
const months = ["Jan", "Febr" , "Mar", Apr ];
const sorted = months.toSorted();
Apr,Feb,Jan,Mar

toReverse method as a safe way to reverse an array without aletring the oroginal array.

Numeric Sort

By default, the sort()  function sorts values as strings.
This works well for strips ("Apple" comes before "Banana").
If numebrs are soreted as strings, "25" is bigger than "100", because "2" is bigger than "1" .
Because of this, the  sort() methods will produce incorrect result when sorting numbers.
You can fix this by prviding a compare function.
Example:
const points = [40, 100, 1, 5, 25, 10];
points.sort(function(a, b){return a - b });
1,5,10,25,40,100

Can be used the same trick to write a descending array:

const points = [40, 100, 1, 5, 25, 10];
points.sort(function(a, b){return b - a });
100,40,25,10,5,1

The compare function

      The purpose of the compare function is to define an alternative sort order.
      The compare function should return a negative, zero or positive value, depending on the arguments:
      function (a, b {return a - b })

      When the sort() functions compare two values, it sends the values to the compare function, 
      and sorts the values according to the returned (negative, zero or positive) value.
      If the result is negative, a  is sorted before  b
    
      If the result is positive, b is sorted before b. 
      If the result is 0, no changes are done with the sort order of the two values.
Example:
      The compare function compares all the values in the array, two values at a time
        (a - b ). 
       When comparing 40 and 100, the  sort() method calls the compare function (40, 100).
       The function calculates 40 - 100 (a - b), and sice the result is negative(-60), the sort function will sort 40 as a value lower than 100.
