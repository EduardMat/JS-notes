JS Array

Why Use Array?

An array is a special variable, which can hold more than one value:
    Example 1:
    const cars =  ["Fiat", "Volvo", "BMW"];
  
This is another way of writing an array:
    Example 2:
    const  cars = [ ];
    cars [0] = "Fiat";
    cars [1] = "Volvo";
    cars [2] = "BMW";
  
    This is another way of creating and array, using the word  new;
    Example 3:
    const  cars = new  Array ("Fiat", "Volvo", "BMW");

1. Accessing an array element

const cars = ["Fiat", "Volvo", "BMW"];
let car = cars[0];
result = Fiat;

2. Changing an element

This statement changes the value of the first element in cars:

const cars = ["Fiat", "Volvo", "BMW"];
cars [0 ] = ["Seat"];
result =["Seat", "Volvo", "BMW"];

3. Converting an array to a string

const fruits = ["Apple", "Orange", "Banana"];
document.getElementById("demo").innerHTML = fruits.to string();

4.Access the Full Array

In JS, the full array can be accessed by referring to the array name:

const cars = ["Fiat", "Volvo", "BMW"];
document.getElementById("demo").innerHTML = cars;
