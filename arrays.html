11.Lopping array elements

One way of looping through an array, is using a for loop:

const fruits = ["Banana","Orange", "Apple", "Mango"];
let fLen = fruits.length;
let text = "ul>";
for ( let i = 0; i < fLen; i++){
text += "li>" + fruits[i] + "/li>";
}
text += "/ul>"
