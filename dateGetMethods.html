The get methods return information from existing date objects. In a date object, the time is static.The "clock" is not "running". The time in a date object is NOT the same as current time.

The fetFullYear() Method

The getFullYear() methods returns the year of a date as a four digit number:

Example:

const d = new Date("2021-03-25");
d.getFullYear();
2021

Example:

const d = new Date();
d.getFullYear();
2024

The getMonth() Method

The getMonth() method returns the moth of a date as a number(0-11)

In JS, January is month 0, February is number 1, and December is number 11.

Example:

const d = new Date("2021-03-25");
d.getMonth();
