JS Functions

A JS function is a block of code designed to perform a particular task.
A function is executed when is invoked.

Example:

Call a function which performs a calculation and returns the result:

script
function myFunction(p1, p2) {
return p1 * p2;
}
let result = myFunction(4, 3);
document.getElementById("demo").innerHTML = result;
script

JS Function Syntax

A JS function is defined with the function keyword, followed by a name, followed by parentesis () .
Function names can contain letters, digits, underscore, and dollar signs (same rules as variables).
The parenthesis may include parameters names separated by commas:(parameter1, parameter2).
The code to be executed, by the function, is placed inside curly brackets.

Example:

function name(parameter1, parameter2, parameter3) {
// code to be executed }

Function parameters are listed inside the paranthesis () in the function definition.
Function arguments are values received by the function when it is invoked.
Inside the function, the arguments (the parameters) behave as local variables.

Function Invokation

The code inside the function will execute when "something" invokes the function:

  • When an event occurs (when a user clicks o button)
  • When it is invoked (called) from JS code
  • Automatically(self- invoked)

Function return

When JS function rechaes a return, the function will stop executing.
If the function was invoked from the statement, JS will "return" to execute the code after the invoked statement.
Functions often compute a return value.The return value is "returned" back to the "caller".

Example:

Call a function which performs a calculation and returns the result:

script
let x = myFunction(4, 3);
document.getElementById("demo").innerHTML = x;
function myFunction(a, b) {
return a * b;
}
script

Why functions?

With functions you can reuse code
You can write code that can be used many times
You can use the same code with different arguments, to produce differner results.

The () Operator

The () operator (calls) the function:

Example:

Convert Fahrenheit to Celsius:

function toCelsius(fahrenheit) {
return (5/9) * (fahrenheit-32);
}
let value = toCelsius(77);



Accessing a function () returns the function and not the function results:

Example:

function toCelsius(fahrenheit) {
return (5/9) * (fahrenheit-32);
}
let value = toCelsius;
