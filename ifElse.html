If, Else and Else if 

     Conditional statements are used to perform different actions based on different conditions.

Conditional statements

Very often when you write code, you want to perform diffrent actions for different decisions.
      You can use conditional statements in your code to do this.
      In JS you have the following conditional statements:
      1.Use if to specify a block of code to be executed, if a specified condition is true.
      2.Use else  to specify a block of code to be executed, if the same condition is false.
      3.Use else if  to specify a new condition to test, if the first condition is false.
      4.Use switch  to specify many alternative blocks of code to be executed.
      The SWITCH statement is described in the next chapter.

1. The switch statement 

Use the if  statement to specify a block of JS code to be executed if a condition is true.
Syntax:
if (condition ) {
  // block of code to be executed if the condition is true
}

Note that if is in lowercase letter.Uppercarse letters (If or IF ) will generate a JS error.
