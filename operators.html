JS Operators

Arithemtics opertors:

IN JS, the following operators do the following:

  • + adds numbers
  • - subtracts numbers
  • * multiply numbers
  • / divides numbers
  • = assigns values

Comparisation operators:

  • 1. = = equal to
  • 2. = = = equal value and equal type
  • 3. ! = not equal
  • 4. ! = = not equal value or not equal type
  • 5. > greated than
  • 6. ( < ) less than
  • 7. >= greater than or equal to
  • 8. <= less than or equal to
  • 9. ? ternary operator

JS string addition

let text2 = "Doe";
let text3 = text1 + " " + text2;

Adding Strings and Numbers

If you are adding numbers to a string, the results will be a string.

Example:

let x = 5 + 5
let y = "5" + 5;
let z = "Hello" + 5;
The result of x, y, and z will be:
10
55
Hello5
