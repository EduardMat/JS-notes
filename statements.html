Java Script Statement

JavaScript is a list of statements to be executed by a computer.

A computer program is a list of "instructions" to be executed by a computer.

In a programming language, these programming instructions are called statements

A JavaScript program is a list of statements (instructions).

JavaScript statements are:

  • Values=> data stored in a variable or constant, can be primitive (a number, string, bolean) and non-primitive (object, functions, arrays).
  • Operators=> are symbols for additions, multiplications, etc (+, *, =)
  • Expressions=> example : 3 + 4 = 7 (3 and 4 are operants, + and = are operators and the 7 is the value)
  • Keywords=> are reserved words which cannot be used to indicate variables(eg. async, let ).There are 63 pf rezerved words.
  • Comments=> usually starting with "//" is a piece of text within a program which is not executed but provide more info about that code

Keywords = reserved words

  • var=> declares a variable
  • let=> declares a block of variable
  • const=> declares a block constant
  • if=> Marks a block of statements to be executed with a condition
  • switch=> marks a block of statements to be executed in different cases
  • for=> marks a block of statements to be executed in a loop
  • function=> declares a function
  • return=> exits a function
  • try=> implement erros handling to a block of statements
