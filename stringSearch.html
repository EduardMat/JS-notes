1. JS Script String indexOf()

The indexOf() method returns the index (position) of the first occurance of a string in a string, or ot returns -1 if the string is not found:
Example:
let text = "Please locate where `locate` is found first time!"
let index = text.indexOf("locate");

2. JS String lastIndexOf()

The lastIndexOf() method returns the index of the last occurrence of a specific text in a string:
Example
let texts = "Please locate where `locate` is found last time!"
let indexs = texts.lastIndexOf("locate");

3. JS String search()

The search() method searches for a string (or a regular expression) and returns the position of the match:
Example
let textes = "Please locate where `locate` is found last time!"
textes.search("locate");

4. JS String match()

The match() method returns an array containing the results of matching a string against a string (or a regular expression).
Example:
let text = "The rain in SPAIN stays mainly in the plain."
text.match("ain");

5. JS String matchAll()

The matchAll() method returns an iterator containing the results of matching a string against a string (or a regular expression).
If the parameter is a regular expression, the global flag (g) must be set, otherwise a TypeError is thrown.
const iterator = text.matchAll(/Cats/g);

Example:
let text = "I love cats. Cats are very easy to love. Cats are nice."
const iterator = text.matchAll("Cats");

6. JS String includes()

The includes() method returns true if a string contains a specific value.
Otherwise it returns false
includes() is case sensitive.

includes() can receive a second parameter.Check if a string includes "world". Start at position 12:
let text = "Hello world, welcome to the universe.";
text.includes("world", 12);
Example:

Check if a string includes "world":

7. JS String startWith()

The startWith() method returns true if a string begins with a specific value.
Otherwise it returns false.
A start position for the search can be specified:
Returns false:
let text = "Hello world, welcome to the universe.";
text.startsWith("world", 5)

Example: let text = "Hello world, welcome to the universe.";
text.startsWith("Hello");

8. JS String endsWith()

The endsWith() method returns true if a string ends with a specified value.
Otherwise returns false .
Example:
let text = "John Doe";
text.endsWith("Doe");
