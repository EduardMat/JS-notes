JS templates

1. Back-ticks syntax

Template strings use back-ticks (` `) rather than quotes to define a string:
Example: let text = `Hello World!`;

2. Quotes Inside String

Template Strings allow both single and double quotes inside a string:
Example:
let texts= `He's often called "Johnny"`;

3. Multiline Strings

Template Strings allow multiline strings:
Example:
let text =
`The quick
brown fox
jumps over
the lazy dog`

4. Interpolation

Template strings provide an easy way to interpolate variables and expressions into strings:
The method is calles string interpolation.
The syntax is :${......}

5. Variable Substitution

Template String allow variables in a string:
Example:
let firstName = "John";
let lastName = "Doe";
let textNew = `Welcome ${firstName}, ${lastName}`;


6. Expression Substitution

Template String llow expressions in string:
Example:
let price = 10;;
let VAT = 0.25;
let total = `Total: ${(price) * (1 + VAT). toFixed(2)}`;
