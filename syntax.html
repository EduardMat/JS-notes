JS Syntax

JavaScript syntax is a set of rules, how JavaScript programs are constructed:

How to create variable:
var x;
var y;

How to use variables:
x = 5;
y = 6;
let z = x + y;

JS Values:

JS syntax defines two types of values:

  • Fixed values=> called Literals(numbers, strings "text")
  • Variable values=>called Variables, used to store data values, for ex: let x = 6; x is assigned(given) the value of 6.

JS Literals

JS Operators=> uses arithmetic operators( +, -, *, / ), numbers and assignment operator (=)

JS Variable

Variables are used to store data values.

JS uses the words var, let and const to declare variables.

An equal sign (=) is used to assign values to variable.

In the example bellow, x is defined as a variable.Then, x is assigned the value of 6:

let x;

x = 6;

JS Expressions:

An expression is a combination of values, variables and operators which compute to a value.
The computation is called an evaluation.
For example, 5 * 10 evaluates to 50.

The values can be of various types, such as numbers or strings.
For ex, "John" + " " + "Doe", evaluates to "John Doe"

JS Keywords

JS keywords are used to identify actions to be performed.
The let keyword tell the browser to create variables.

JS Comments

There are things which are not executed, if ther are placed between "//" or /*...*/.These are called comments.

JS identifiers

Identifiers are used to name variables and keywords, and functions.The rules for legal names are the same in most programming languages.A name is not allowed to start with a number.Also, remember that JS is case sensitive.lastName is different of lastname.
A JS name bust begin with:

  • A letter ( A-Z or a-z)
  • A dollar sign ($)
  • Or an underscore(_)

JS and Camel Case

Programmers have used diferrent \\ways of joining words, like "first-name" using hyphens(-), however this is not allowed in JS, - being used for subtractions.
Could be used instead Upper Camel Case(Pascal case), eg: FirstName, LastName, InterCity.
Alterantively, can be used Lower Camel Case, eg : firstName, lastName, interCity

JS Character Set

JS uses the Unicode character set - which covers all the characters, punctuation, and symbols in the world.
