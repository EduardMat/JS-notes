JS Identifiers

All JS variables must be identified with UNIQUE NAMES

The unique names are called identifiers.

Identifiers can be short names (like x, y) or be more descriptive, like age, sum, totalVolume.
The general rules for constructing names are:

  • 1.Names can containers letters, digits, underscores, and dollar sign
  • 2.Names must begin with a letter
  • 3.Names can also begin with $ and _
  • 4.Names are case sensitive
  • 5.Reserved words cannot be used as names

JS Dollar sign $

Since JavaScript treats a dollar sign as a letter, identifiers containing $ are valid variable names:

Example:

let $ = "Hello World";
let $$$ = 2;
let $myMoney = 5;

JS Underscore _

Since JavaScript treats underscore as a letter, identifiers containing _ are valid variable names.
let _lastName = "Johnson";
let _x = 2;
let _100 = 5;
